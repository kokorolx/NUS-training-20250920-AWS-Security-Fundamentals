Query examples for security analysis

  • Failed SSH logins (24h): count by src_ip where event='FailedLogin' and timestamp>now-24h
  • CloudTrail CreateUser/Policy: filter eventName in (CreateUser,PutUserPolicy) | where sourceIP not in (expected_ips)
  • S3 public buckets: config.resourceType='AWS::S3::Bucket' and (bucketPolicyPublic=true or aclPublic=true)
  • Data exfil pattern: sum(bytes) by user where action in (GetObject,Download) and bytes > 10MB
  • Azure KQL (4625): SecurityEvent | where EventID==4625 | summarize count() by Account, IPAddress
