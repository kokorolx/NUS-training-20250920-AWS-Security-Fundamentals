Real-world Security Incidents

🚨 Capital One (2019)

Nguyên nhân: Misconfigured web application firewall

Tác động: 100 triệu hồ sơ khách hàng bị lộ

Chi phí: $300 triệu USD

Bài học: Cấu hình bảo mật đúng cách

🔓 Uber (2016)

Nguyên nhân: API keys leaked in GitHub

Tác động: 57 triệu tài khoản bị xâm phạm

Chi phí: $148 triệu USD

Bài học: Quản lý credentials an toàn
